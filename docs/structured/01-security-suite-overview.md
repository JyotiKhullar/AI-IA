# Security Suite Overview

The **Security Suite** in *GatewayX* includes:
- **Multi-Factor Authentication (MFA)**
- **Single Sign-On (SSO)**
- **Roles & Permissions**

## Key Capabilities
- Strong authentication using MFA.
- Centralized identity via SSO.
- Granular access control with roles.
